# E. Data Visualization Summary

This folder uses `ggplot2` to turn `mtcars` into simple charts. The scripts save PNG files in `output/` so the graphs can be checked and submitted.

## `data_visualization_1.R`

This script makes two charts: a histogram for `mpg` and a bar plot for cylinder counts. Both are saved as PNG files in the output folder.

## `data_visualization_2.R`

This script makes two more charts: a scatter plot for weight versus mileage and a boxplot for mileage by cylinder group. Both are also saved as PNG files.
