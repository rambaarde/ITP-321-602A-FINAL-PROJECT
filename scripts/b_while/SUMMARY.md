# B. While Loop Summary

This folder shows how a `while` loop keeps running as long as a condition stays true. It is useful when the ending point depends on the data.

## `while_1.R`

This script counts from 1 to 5 with a `while` loop. The counter increases one step at a time until it reaches the limit, then the full sequence `1 2 3 4 5` is printed.

## `while_2.R`

This script keeps sampling numbers until the running mean reaches 5. It stores each draw, recalculates the mean after every step, and prints the final sample path and mean.
