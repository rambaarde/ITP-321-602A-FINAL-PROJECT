# B. While Loop

A `while` loop keeps running while a condition stays true. Use it when the stop point is not fixed yet.

## Scripts

- `while_1.R` - count from 1 to 5
- `while_2.R` - sample until running mean reaches 5

## Run

```r
source("final-proj/scripts/b_while/while_1.R")
source("final-proj/scripts/b_while/while_2.R")
```

## Expected Output

- `while_1.R` prints `1 2 3 4 5`
- `while_2.R` prints the sample vector and final mean
