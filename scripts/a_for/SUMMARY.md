# A. For Loop Summary

This folder shows how a `for` loop moves through a fixed set of values one item at a time. It is useful when the number of steps is already known.

## `for_1.R`

This script adds the values in a numeric vector with a `for` loop. The loop takes each number, adds it to a running total, and prints the final answer, which is `30`.

## `for_2.R`

This script uses a `for` loop to calculate the mean of selected `mtcars` columns. It checks `mpg`, `hp`, and `wt` one by one, then prints a named result with the average for each column.
