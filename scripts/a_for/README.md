# A. For Loop

A `for` loop runs one step per item in a vector. Use it when you already know the full set of values.

## Scripts

- `for_1.R` - sum a numeric vector
- `for_2.R` - compute means for selected `mtcars` columns

## Run

```r
source("final-proj/scripts/a_for/for_1.R")
source("final-proj/scripts/a_for/for_2.R")
```

## Expected Output

- `for_1.R` prints `30`
- `for_2.R` prints named means for `mpg`, `hp`, and `wt`
