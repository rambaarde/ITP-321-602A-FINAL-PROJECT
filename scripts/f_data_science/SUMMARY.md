# F. Data Science Summary

This folder shows a small data analysis workflow with `dplyr` and base R modeling. The first script summarizes the data, and the second script fits a simple model.

## `data_science_1.R`

This script groups `mtcars` by cylinder count and calculates average `mpg`, `hp`, and `wt` for each group. The result is a small summary table printed in the Console.

## `data_science_2.R`

This script fits a linear model that predicts `mpg` from `wt` and `cyl`. It prints the model summary so the fit and coefficients can be checked in the Console.
