# D. Functions

Functions make code reusable. Use them when the same logic should run more than once.

## Scripts

- `functions_1.R` - multiply two numbers
- `functions_2.R` - summarize a numeric vector

## Run

```r
source("final-proj/scripts/d_functions/functions_1.R")
source("final-proj/scripts/d_functions/functions_2.R")
```

## Expected Output

- `functions_1.R` prints `20` and `21`
- `functions_2.R` prints summary stats for `mpg` and `hp`
