# D. Functions Summary

This folder shows how functions let you reuse the same logic in more than one place. Each script defines a function, calls it, and prints the result.

## `functions_1.R`

This script defines a multiplication function called `multiply_numbers()`. It takes two numbers, multiplies them, and prints test results of `20` and `21`.

## `functions_2.R`

This script defines a summary function called `summarize_vector()`. It returns basic stats like count, mean, median, standard deviation, min, and max, then prints summaries for `mtcars$mpg` and `mtcars$hp`.
