# C. Repeat Loop

A `repeat` loop keeps going until `break` stops it. Use it when the loop ends based on a custom condition.

## Scripts

- `repeat_1.R` - stop when sampled value reaches threshold
- `repeat_2.R` - retry until random success

## Run

```r
source("final-proj/scripts/c_repeat/repeat_1.R")
source("final-proj/scripts/c_repeat/repeat_2.R")
```

## Expected Output

- `repeat_1.R` prints sampled values
- `repeat_2.R` prints a list with `success` and `attempts`
